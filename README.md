Trello UI Automation Project

Overview
This is an automated test script written in Python using Selenium to verify the accessibility of the Trello platform.

Features
-Automated Browser Control: Uses Selenium to launch Chrome.
-Dynamic Driver Management: Uses `webdriver-manager` to handle browser versions.
-Sync Handling: Implements time delays to ensure page loading.

How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the script: `python test_trello.py`
